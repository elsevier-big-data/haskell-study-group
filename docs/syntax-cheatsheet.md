### Syntax

##### Function
```haskell
function_name input_parameter1 input_parameter2 input_parameterN = result
```

##### Immutable Value
```haskell
x = 1
```

##### Variable
```haskell
let x = 1
```

##### If Then Else
```haskell
calcChange change = if change > 0
                        then change
                        else 0
```

##### Where
```haskell
doublePlusTwo x = doubleX + 2
   where doubleX = x * 2
```

##### Lambda Function
```haskell
(\x -> x) argToPass
```

##### List 
```haskell
x = ["a", "b"]
```
